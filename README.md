## Installation  

1. **Clone the repo**  
   ```bash
   git clone https://github.com/harshshubham987-svg/harsh-portfolio.git
   cd harsh-portfolio
   ```

2. **Install dependencies**  
   ```bash
   npm install
   ```

3. **Set up environment variables** (if any) – create a `.env` file in the root.  
   Example:  
   ```env
   VITE_API_ENDPOINT=https://api.example.com
   ```

---

## Run Locally  

```bash
npm run dev
```
Open **http://localhost:5173** in your browser. The app will hot‑reload on changes.

---

## Build for Production  

```bash
npm run build
```
The production build lands in the `dist/` folder. Deploy the contents of `dist/` to any static host (e.g., Vercel, Netlify).

**Deploy to Vercel** (quick start):
```bash
npm run vercel -- --prebuilt
```
